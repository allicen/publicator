О программе

Программа «Публикатор» предназначена для публикации на сайте статей из Word-файлов.

Программа конвертирует .docx-файл в формат html, извлекает изображения, формирует заголовок H1 и ссылку на статью.

Полученные данные можно отредактировать в интерфейсе "Публикатора", добавить анонс, мета-теги, загрузить главное изображение записи.

Важно:

  • Конвертирует только файлы формата .docx.
  • Не учитывает объединение ячеек при формировании таблиц. Не нужно загружать таблицы с объединенными ячейками!
  • Абзац, в котором встречается курсив, преобразуется в блок цитирования.
  • Не учитывает копии одной и той же картинки в одном Word-документе. Сохраняет только одну картинку.
