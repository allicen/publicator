Инструкция по работе с программой

  1. Настройка программы
  2. Порядок работы с программой

1. Настройка программы

Запустите программу. Появится интерфейс:

Перейдите в раздел «Настройки / Настройки импорта».

В разделе «Параметры» отметьте чек-боксы настроек импорта, которые обязательно должны быть заполнены перед публикацией записи.

Укажите директорию для хранения изображений на сервере.

Во вкладках «FTP», «SSH», «MySQL» заполните все настройки. И сохраните их.

2. Порядок работы с программой

Откройте конвертер.

Выберите подготовленный Word-файл.

Если файл будет загружен, вы увидите сообщение об успешной загрузке.

Перейдите на соседнюю вкладку «Предпросмотр». Это можно сделать, кликнув по вкладке, или по кнопке «Далее».

Откроется предпросмотр документа.

Проверьте подготовленный html. Если необходимо внести правки в документ, то можно кликнуть на кнопку «Править» или перейти на вкладку «HTML-редактор».

Если правок не требуется, то можно кликнуть по кнопке «Всё OK!» и тогда откроется вкладка «Дополнительно».

Во вкладке «HTML-редактор» можно исправить полученный документ в поле textarea или в визуальном редакторе. Правки из textarea можно передать в визуальный редактор с помощью кнопки «Из HTML». Правки из визуального редактора можно передать в поле textarea с помощью кнопки «В HTML». Итоговый текст статьи берется из поля textarea. Кнопка «Очистить всё» сбрасывает загруженный документ (очищаются все поля и папки, которые были связаны с загрузкой).

На вкладке «Дополнительно» указываются дополнительные настройки. Здесь можно заполнить недостающие данные и прикрепить главную картинку записи.

После загрузки файла, в котором был указан заголовок H1, эта вкладка выглядит вот так:

Можно заполнить недостающие поля и прикрепить фото.

На вкладке «Публикация» нужно получить категории из базы данных.

Если среди полученных категорий нет нужной, то можно добавить свою.

Категория будет добавлена.

Завершающий этап – проверка параметров публикации.

Появляется сообщение о заполненности полей. Если статья готова к публикации, то кнопка «Опубликовать» становится активной.

Статья появится на сайте.
